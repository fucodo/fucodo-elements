## Description

This page documents the Bootstrap-flavored usage of the `image-crop-dialog` web component. It showcases:

- A Bootstrap-style file input (`class="form-control"`).
- Localized action labels provided via `label-cancel` and `label-confirm` slots, each including an inline SVG icon.
- Form integration: on submit, the cropped image is placed in the form data under the input's `name`.

Tag name: `image-crop-dialog`

## Notes

- Put your `<input type="file">` inside the dialog; the component will use it as the image source.
- You can localize the action buttons using `label-cancel` and `label-confirm` slots and include icons as shown above.
- The resulting cropped file will be available in the `FormData` under the input's `name` (e.g., `object[passbild]`).
