## Description

The `image-crop-dialog` web component opens a dialog that lets users select and crop an image
to a specific target size. It is designed to be embedded in forms: once the user confirms the crop,
the resulting image file is placed in the form data under the same `name` as the file input.

This documentation page is powered by the same stories and args used in the controls panel,
so what you see here always reflects the live story configuration.

Tag name: `image-crop-dialog`

## Notes

- Place an `<input type="file" name="...">` inside the component; it will be used as the source image.
- Provide optional slot content for `label-cancel` and `label-confirm` to localize the dialog actions.
- On submit, the cropped image will be available from the form data for the file input's `name`.
