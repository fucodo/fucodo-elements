## Description

The `fucodo-tabs` web component enhances standard Bootstrap tabs. It provides two main features:
1. **Persistence**: It automatically saves the active tab in `localStorage` and restores it on page reload.
2. **Validation Error Highlighting**: It scans tab panels for elements with the `.is-invalid` class and applies a `text-bg-danger` class to the corresponding tab button.

Tag name: `fucodo-tabs`

## Usage

The component expects a standard Bootstrap tab structure inside it: a `.nav-tabs` container and corresponding `.tab-pane` elements.

<Source
  language="html"
  code={`<fucodo-tabs key="my-unique-tabs">

## Properties / Attributes

- `key` (string): Used as a salt for the storage key. If not provided, `location.pathname` is used.
- `state-key` (string): Optional additional key to further differentiate the storage key.

<ArgTypes of={TabsStories.Default} />

## How it works

- **Storage Key**: The key used in `localStorage` is generated by hashing the combined text of all tab links, the `key` attribute (or pathname), and the `state-key`.
- **Restoration**: On connection, it checks `localStorage` for a saved target selector. If found, it attempts to show that tab using Bootstrap's Tab API or a manual fallback.
- **Validation**: It periodically checks (on connection) if any `.tab-pane` contains `.is-invalid` elements. If so, it adds `text-bg-danger` to the tab link.
